# Control-TV
Python script to control tv's movement via serial connection through jig

Before you continue with this program, install the following libraries:
1. pyserial
2. click

Input below commands in command prompt (cmd) to install libraries:
pyserial: "pip install pyserial"
click   : "pip install click"

How to run:
1. Connect JIG from your laptop to the TV
2. Open CMD/Terminal
3. run: python Control_TV.py

Supported TVs:
1. Bluefin
2. Chutoro
3. Vulcan
4. Amaebi
